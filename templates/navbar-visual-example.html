📋 Navigatiebar Template

Overzicht

De navigatiebar staat bovenaan elke pagina en bevat het site-logo/titel aan de linkerkant en navigatielinks aan de rechterkant. Het design volgt Beautiful Jekyll's stijl met een crème-transparante achtergrond en glasmorphism effect.

🎨 Stijl Eigenschappen

Kleuren

Primair: #2f6f3e (Groen)
Donker: #1f4f2d (Donkergroen)
Tekst: #263126 (Donkerbruin)
Achtergrond: rgba(247, 243, 236, 0.92) (Crème transparant)

Typografie

Font: Source Sans 3
Titel Gewicht: 800 (Bold)
Link Gewicht: 500 (Medium)
Titel Grootte: 20px
Link Grootte: 15px

💻 HTML Structuur

<nav class="navbar-custom"> <div class="navbar-inner"> <a class="navbar-brand" href="#">Roadtrips</a> <ul class="navbar-nav"> <li><a class="nav-link active" href="#">Campings</a></li> <li><a class="nav-link" href="#">Fietstochten</a></li> <li><a class="nav-link" href="#">Wandelroutes</a></li> <li><a class="nav-link" href="#">Plaatsen</a></li> <li><a class="nav-link" href="#">Musea</a></li> <li><a class="nav-link" href="#">Kano & SUP</a></li> </ul> </div> </nav>

📐 Layout Eigenschappen

Positionering

Position: Sticky (blijft bovenaan tijdens scrollen)
Z-index: 20 (vooraan andere elementen)
Padding: 18px 24px
Max-width: 1200px (inhoud gelimiteerd)

Interactie

Hover effect: Link kleur verandert naar groen (#2f6f3e)
Active state: Groene kleur + onderline
Transition: 200ms ease
Responsive: Hamburger-menu op <768px

🔧 CSS Classes

.navbar-custom /* Main container */ .navbar-inner /* Inner wrapper, flex */ .navbar-brand /* Site title/logo */ .navbar-nav /* Navigation list */ .nav-link /* Individual nav link */ .nav-link.active /* Active link state */ .nav-link:hover /* Hover state */

📱 Responsive Gedrag

Op schermen smaller dan 768px verdwijnt de navigatiebar en verschijnt een hamburger-menu icon. Dit wordt beheerd door Beautiful Jekyll's Bootstrap navbar-toggler component.
